# Basic Recon using Nmap

 Overview:
This project demonstrates basic network scanning techniques using Nmap.

 Objective:
To understand how to discover open ports and services on a target system.

 Tool Used:
- Nmap

 Commands Used:
- nmap -sS <target>
- nmap -sV <target>

 Findings:
- Identified open ports
- Detected running services

 Learning Outcome:
- Learned basic reconnaissance techniques
- Understood how attackers gather information
